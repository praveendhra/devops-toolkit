# DevOps Toolkit

A collection of reusable DevOps templates, configurations, and scripts for CI/CD, containerization, orchestration, and monitoring.

## Structure

```
├── github-actions/       # GitHub Actions workflow templates
├── azure-pipelines/      # Azure DevOps pipeline templates
├── docker/               # Dockerfiles and compose configs
├── kubernetes/           # K8s manifests and Helm values
├── scripts/              # Operational shell scripts
├── monitoring/           # Prometheus, Grafana, alerting configs
├── nginx/                # NGINX reverse proxy configurations
└── security/             # Security scanning and policy configs
```

## Usage

Each directory contains ready-to-use templates. Copy what you need and customize for your project.

## Tech Stack

- **CI/CD**: GitHub Actions, Azure DevOps
- **Containers**: Docker, Docker Compose
- **Orchestration**: Kubernetes
- **Monitoring**: Prometheus, Grafana
- **Proxy**: NGINX
- **Cloud**: AWS, Azure, GCP
